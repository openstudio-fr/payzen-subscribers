Pay a subscription or donation with PayZen. 

== Description ==

This plugin does :
- Config payzen API
- Create multiple plans of subscriptions
- One client payment
- One client subscription 
- Add widgets to subscribe
   - Change user if subscribed or not
   - Change payment method
   - Display invoices
